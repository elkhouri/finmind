## FinMind Dashboard

Run the development server with `pnpm dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Deployed to [https://finmind-drab.vercel.app/](https://finmind-drab.vercel.app/). May not work 100% of the time due to Vercel and FinMind free limits.

## Notes
- Tech Stack: Typescript, React, Next.js
- The component library used is MUI, with Tailwind CSS to supplement it.
- Theming is mostly done in `app/globals.css`, with some MUI customizations in `theme.ts`.
- State management is done via Zustand.
